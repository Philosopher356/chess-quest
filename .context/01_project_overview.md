# Chess Quest - Project Overview

## What Is This
A Flutter chess learning app for kids (ages 5+) that teaches chess from absolute beginner to master level through interactive, hands-on lessons with gamification. Runs on web, Android, and iOS.

## User Requirements (from initial request)
- Systematically teach children to play chess in a playful and interactive manner
- Rewards along the way
- All basic and strategic concepts up to master level
- Interactive and hands-on (NOT theoretical explanations)
- Each concept must be exercised interactively
- Built in Flutter for web, Android, and iOS

## User Decisions
- **Distribution**: All three platforms - Play Store, App Store, and Web
- **Audio**: Use free sound libraries (ended up generating synthesized WAV files with Python)
- **Testing**: Core logic tests only (highest ROI)
- **CI/CD**: GitHub Actions

## App Name
**Chess Quest** (bundle ID: `com.chessquest.chess_quest`)

## Mascot
**Sir Hops** - a friendly knight character that guides children through lessons
